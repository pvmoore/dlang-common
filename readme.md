# Dlang Common Utilities Library

#### Features
- Allocator
- Array
- bool3
- List (A single linked list)
- ObjectCache
- PDH (Windows performance data helper)
- Pinned Array (wip)
- Queue
- Set
- Stack
- StringBuffer
- StructCache
- TreeList
- Velocity (A simple templating system a la Velocity but not Velocity. Not
    anywhere near as full featured)

###### Async 
- AsyncArray (Locking array)
- AsyncQueue (Non-locking queue)

###### IO 
- BitReader
- BitWriter
- ByteReader
- ByteWriter

###### Other 
- array_utils - built-in array UFCS
- map_utils - associative array UFCS
- static_utils - static template functions
- string_utils - string UFCS
- timing - Timing utility
- utilities - Small miscelaneous utilities

#### Requirements
- Dlang https://dlang.org/
